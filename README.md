# RepositorioTemplate

Repositório que deve ser utilizado como template inicial pelos grupos da matéria de Arquitetura e Desenho de Software.

## Link do deploy da aplicação

Acesse a aplicação pelo link: [QueroBemEstar](https://2025-2-t01-g6-quero-bem-estar-entre.vercel.app/)

## Introdução

Este repositório traz um template de repo de documentação a ser seguido pelos grupos de arquitetura e desenho de software.

## Tecnologia

A geração do site estático é realizada utilizando o [docsify](https://docsify.js.org/).

### Instalando o docsify

Execute o comando:

```shell
npm i docsify-cli -g
```

### Acessando pela Internet

Para acessar a página web da nossa documentação [Clique aqui](https://unbarqdsw2025-2-turma01.github.io/2025.2-T01-G6-QueroBemEstar_Entrega_04/#/)

### Executando localmente

Para iniciar o site localmente, utilize o comando:

```shell
docsify serve ./docs
```
